# MediFlow

MediFlow is a Hospital Management System built using Spring Boot and MySQL.

## Features Implemented

### Patient Management

* Create Patient
* Get Patient Details
* Update Patient Information
* Delete Patient

### Doctor Management

* Create Doctor
* Get Doctor Details
* Update Doctor Information
* Delete Doctor

### Exception Handling

* Global Exception Handler
* Custom Exceptions

## Tech Stack

* Java 21
* Spring Boot
* Spring Data JPA
* MySQL
* Maven
* Lombok

## Project Structure

* Controller Layer
* Service Layer
* Repository Layer
* Entity Layer
* Exception Handling Layer
